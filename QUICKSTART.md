# Quick Start Guide

**Toyota GR Racing Telemetry System**  
**Developer: Tanush Shah (The Unknown God)**

## Get Started in 3 Steps

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Open in Browser
Navigate to `http://localhost:5173`

## Using the System

### Select Data Source
1. Choose a track from the dropdown (7 circuits available)
2. Select Race 1 or Race 2
3. Optional: Enter vehicle number to filter
4. Click "LOAD RACE DATA"

### View Analysis
- **3D Visualization** - Track replay with animated cars
- **Race Insights** - AI-powered performance analysis
- **Strategy & Predictions** - ML-based forecasting

## Key Features

- Real telemetry from 7 professional circuits
- Automatic corner detection
- AI-powered insights
- ML lap time predictions
- 3D track visualization
- 15+ technical charts

## Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder.
